## Блок 1: Три типа инструментов

Инструментов для вайбкодинга много. Но они делятся на три типа. Не нужно знать все — нужно знать по одному из каждого типа.

### Тип 1: ИИ-кодеры (пишут код за вас)

Это ваш основной рабочий инструмент. Вы описываете задачу — они создают готовый код.

**Claude Code** — расширение для VS Code от компании Anthropic.

Что делает: всё. Сайты, боты, приложения, автоматизации, базы данных, деплой.
Где работает: внутри VS Code (редактора кода).
Сильные стороны: видит весь проект целиком, работает с любыми файлами, полный контроль.
Слабые стороны: платный, нужна установка VS Code.
Стоимость: Pro — $20/мес (~2 000 ₽), Max — $100–200/мес.

**Cursor** — отдельный редактор кода со встроенным ИИ.

Что делает: то же, что Claude Code, но в собственном редакторе.
Где работает: отдельная программа (похожа на VS Code).
Сильные стороны: красивый интерфейс, удобный для новичков.
Слабые стороны: платный, ограничение по количеству запросов.
Стоимость: Pro — $20/мес (~2 000 ₽).

### Тип 2: ИИ-билдеры (строят в браузере)

Не нужно ничего устанавливать. Открыли браузер — и работаете.

**Lovable** — создание сайтов и приложений прямо в браузере.

Что делает: полноценные веб-приложения — фронтенд, бэкенд, база данных, хостинг.
Где работает: в браузере — lovable.dev
Сильные стороны: результат за минуту, мгновенный хостинг, красивые интерфейсы.
Слабые стороны: только веб-сайты, токены сгорают быстро.
Стоимость: бесплатно (5 генераций/день), Pro — $25/мес.

> Попробовать Lovable: https://lovable.dev/invite/I9HLR65

### Тип 3: ИИ-чаты (для вопросов и текста)

Нужны для подготовительной работы: продумать структуру, написать текст, разобраться в задаче.

**Google AI Studio** — бесплатный ИИ от Google.

Что делает: отвечает на вопросы, генерирует тексты, помогает с идеями.
Где работает: в браузере — aistudio.google.com
Сильные стороны: бесплатный, работает без ограничений по региону, мощная модель Gemini.
Слабые стороны: не создаёт файлы, код нужно копировать вручную.
Стоимость: бесплатно (1 000 запросов на Gemini).

### Какой инструмент выбрать для старта?

| Ситуация | Рекомендация |
|----------|-------------|
| Хочу попробовать бесплатно | Google AI Studio + Lovable (бесплатный план) |
| Готов вложить 2 000 ₽/мес | Claude Code (Pro) или Cursor (Pro) |
| Хочу делать сайты быстро | Lovable |
| Хочу делать всё (сайты + боты + автоматизации) | Claude Code + VS Code |

<Callout type="tip">
Начните с бесплатных: Google AI Studio для вопросов, Lovable для первых сайтов. Когда поймёте, что это ваше — переходите на Claude Code или Cursor.
</Callout>

<Callout type="success" title="Что поняли">
Три типа инструментов: кодеры (Claude Code, Cursor), билдеры (Lovable), чаты (Google AI Studio). Для старта хватит одного из каждого типа. Можно начать полностью бесплатно.
</Callout>

## Блок 2: Дополнительные инструменты

Помимо ИИ, вам понадобятся ещё несколько вещей. Все они бесплатные или почти бесплатные.

### VS Code — ваша мастерская

Бесплатный редактор кода от Microsoft. Это рабочее место, где вы будете проводить большую часть времени.

Скачать: code.visualstudio.com

Установка за 2 минуты:
1. Скачайте с сайта (Mac или Windows)
2. Установите (перетащите в папку «Программы» на Mac или нажмите «Далее» на Windows)
3. Откройте

### GitHub — ваш портфель работ

Облачное хранилище для проектов. Аналогия: Google Диск, но для кода. С историей изменений.

Зачем нужен:
- Сохранять проекты (как «сохранение» в игре)
- Показывать работы клиентам
- Откатываться, если что-то сломалось

Зарегистрироваться: github.com (бесплатно)

### Vercel — бесплатный хостинг

Платформа для публикации сайтов. Сделали сайт — загрузили на Vercel — он в интернете.

Зачем нужен:
- Бесплатный хостинг для сайтов
- Автоматический HTTPS (безопасное соединение)
- Подключение своего домена

Зарегистрироваться: vercel.com (бесплатно, привязать GitHub)

### Домен — адрес в интернете

Домен — это адрес вашего сайта. Например, moysayt.ru.

Где купить:
- **reg.ru** — российский регистратор, оплата в рублях
- **timeweb.ru** — ещё один российский, удобный и недорогой
- **Namecheap** — международный, в долларах
- **Cloudflare** — без наценки, по себестоимости

Стоимость: от 500 ₽/год за зону .ru

<Callout type="success" title="Что поняли">
Кроме ИИ нужны: VS Code (мастерская), GitHub (портфель), Vercel (хостинг), домен (адрес). Всё бесплатно, кроме домена (~500 ₽/год).
</Callout>
